Super vs Hyper

Supercars

What is a Supercar: 

    We would consider the definition of a supercar to be an eye catching
    factory stock car which accelerates faster and is more expensive
    than at least 98% of the other cars manufactured in the same year.

Hypercars

The term hypercar seems to be used more and more each year.
We simply define a hypercar as a top-tier supercar that bests all or most of
the elements of standard supercars including, but not limited to, acceleration,
price, rarity, handling, top speed, sound and appearance.
